## Pasos para replicar la prueba

### clonar el proyecto

git clone https://github.com/kelvinsinsua/problema2-reactjs-frontend.git

### instalar las dependencias

cd problema2-reactjs-frontend && yarn install

### iniciar dev server

yarn start

### compilar para producción

yarn build
